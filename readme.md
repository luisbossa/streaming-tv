# InstaTV - Streaming Web App

InstaTV es una aplicación web de streaming de canales en vivo desarrollada con HTML, CSS y JavaScript puro. Permite reproducir transmisiones HLS (.m3u8), filtrar canales por categorías y controlar la reproducción mediante un reproductor personalizado tipo TV.

---

## Características principales

- Reproducción de canales en vivo mediante HLS (hls.js)
- Sistema de categorías:
  - Cine
  - Deportes
  - Música
  - Noticias
  - Shows
  - Reportajes
  - Caricaturas
- Filtro dinámico de canales por categoría
- Controles de reproducción (Play / Pause)
- Control de volumen con slider
- Modo pantalla completa
- Canal activo resaltado en la interfaz
- Reproducción automática del primer canal
- Contador de canales totales y en línea

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- hls.js
